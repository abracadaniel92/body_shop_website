# Platform Decision

## Chosen stack

- CMS: WordPress
- Frontend approach: Custom WordPress theme
- Hosting target: Managed WordPress host or VPS with backups/SSL/staging

## Why this matches non-technical users

- Editing is done in a visual admin UI.
- Blog publishing is native and familiar.
- Image uploads are drag-and-drop in media library.
- No code deploy is required for normal content updates.

## Scope covered

- One-page marketing homepage (hero, services, contact, nav, logo).
- Gallery page with before/after uploads from admin.
- Blog page with post listing and detail pages.
