# Editor Guide (Non-Technical)

This guide covers the most common tasks without touching code.

## 1) Update homepage content

1. Login to WordPress admin (`/wp-admin`).
2. Go to `Pages` -> `Home`.
3. Edit text blocks and section content.
4. Click `Update`.

## 2) Add a before/after gallery item

1. Go to `Gallery Items` -> `Add New`.
2. Enter title and optional description.
3. Upload `Before` and `After` images in `Media`.
4. Copy each attachment ID (from media item URL or details plugin).
5. Paste IDs into `Before/After Details` fields.
6. Click `Publish`.

## 3) Publish a blog post

1. Go to `Posts` -> `Add New`.
2. Add title, content, and featured image.
3. Select category/tags.
4. Click `Publish`.

## 4) Update menu links

1. Go to `Appearance` -> `Menus`.
2. Edit the `Primary Menu`.
3. Ensure Home, Gallery, Blog, and Contact are listed.
4. Save menu.

## 5) Safe publishing checklist

- Preview page/post before publishing.
- Confirm image quality on mobile and desktop.
- Check phone/email links in contact section.
- Avoid deleting existing gallery entries unless intentional.
